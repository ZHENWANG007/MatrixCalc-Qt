# MatrixCalc-Qt
一个基于 Qt (C++) 开发的轻量级矩阵计算工具，支持矩阵加法、乘法、转置等基础运算，搭配直观的可视化界面，操作简单且跨平台。

## 功能特性
-  核心运算：支持矩阵加法、乘法、转置三大基础操作
-  可视化交互：通过表格界面直接输入/输出矩阵数据，无需手动编写数据格式
-  异常防护：内置维度不匹配、输入非法值等校验，弹窗提示错误信息，避免程序崩溃
-  跨平台运行：适配 Windows/macOS/Linux 系统

## 构建要求
- CMake 3.16 及以上版本
- Qt 5.15+ / Qt 6.0+（需包含 Qt Widgets 模块）
- 支持 C++17 的编译器（GCC/Clang/MSVC 均可）

## 快速构建 & 运行
```bash
# 创建并进入构建目录
mkdir build && cd build

# CMake 配置
cmake ..

# 编译项目（Release 模式）
cmake --build . --config Release

# 运行程序（Linux/macOS）
./MatrixCalc

# 运行程序（Windows）
Release\MatrixCalc.exe
```

## 使用说明
1. 启动程序后，选择矩阵的行/列数，点击「创建矩阵」初始化矩阵 A/B
2. 在矩阵 A/矩阵 B 表格中输入数值
3. 点击对应按钮执行运算：
   - 「加法」：需保证两个矩阵维度完全一致
   - 「乘法」：需保证矩阵 A 的列数 = 矩阵 B 的行数
   - 「转置」：仅对矩阵 A 进行行列互换操作
4. 运算结果会实时显示在「结果」表格中
